Manage and Pay your Parking Tickets on the go.

Branding

The goal of the logo was to convey that the product was all about Parking Tickets. After a few iterations and tests we decided to use the Parking Sign design because the user could assimilate our product with Parking right away.

UX - User Experience

A tool provides a good user experience if it is Understandable, Actionable, Engaging, and Efficient.

Pay Everywhere

One of the main goals at FineNYC was for the user to be able to pay on any device, so a responsive design was essential.

Pay by Phone

UX tests showed us that our older users couldn’t see the screen properly, so our solution was to launch an option to pay by simply calling to our automated voice system.

Website Design

Responsive

One of the most appealing aspects of responsive web design is that a responsive website can provide a great user-experience across many devices and screen sizes. This is an important characteristic, since it is impossible to anticipate all the devices and screen sizes users will use to access the website.
